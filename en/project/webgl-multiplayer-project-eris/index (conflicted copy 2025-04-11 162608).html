WebGL Multiplayer Project Made In Unity

  • January 4, 2025
Environment & Level Design Intern → Art Direction Assistant | 2022–Present
Unity (URP), WebGL

Project Eris is a large-scale multiplayer online role-playing game developed in Unity using the Universal Render Pipeline (URP). The project’s vision is to revive and evolve the design philosophies of early 2000s classic MMOs, delivering a nostalgic yet modern gameplay experience.

Art Direction

As the project progressed, I moved from environment and level design into a supporting role in art direction.

The visual style draws from the realism of early 2000s MMOs — a style once seen as realistic, but now perceived as stylized. I contributed 3D character models and visual references aligned with this direction, focusing on a dark fantasy tone that avoids modern hyper-realism.

All assets were designed with performance in mind, supporting lower-end systems and WebGL deployment.

UI Design

I designed the game’s user interface to match its visual identity—merging dark fantasy themes with modern, minimalist elements. The interface was designed with clarity and coherence within a browser-based environment in mind.

Terrain Workflow & Tools

To support large-scale worldbuilding, I developed a procedural terrain generation workflow using World Machine. I generated heightmaps and splatmaps, split them into chunks, and used Unity’s terrain tools to construct the in-game environment. This workflow was designed to make future updates and modifications more efficient.

I also implemented a custom object painting tool to streamline asset placement and accelerate level design.

Large structures were visually integrated using custom decals to ensure consistency across the environment.
